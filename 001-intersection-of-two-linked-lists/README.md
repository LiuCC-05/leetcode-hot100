题目描述：给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 null 
关键知识：1.针对链表，我们只要知道head（头指针）就可以得到整个链表的信息，包括链表长度/链表内容/（通过遍历得到）
          2.链表长度，通过head.next判断不为None，然后count+=1得到
两种解决方法
统一需要判断headA is not None or headB is not None 否则 return None
第一种：长度差法
      对于两个链表，在正式相交之前，存在不公共的部分，对两链表长度进行做差，得到diff值，让长的链表先提前走diff步，得到长链表走后的head，然后再同时让两个head（短链此时的head就是最初的head）向下遍历，则可得到下一个节点就是相交节点的位置；
第二种：双指针法
      依靠一个思路：两个指针走相同的步数之后，两个指针指向的下一个节点将相同
      理解思路：移动的总距离=A独有+公共部分+B独有
      pA指针遍历完A后遍历B
      pB指针遍历完B后遍历A
      循环对每个指针当下指向的节点进行is判断，最后都会走到相交节点位置或者None
      ！当两链表没有公共部分是：两指针遍历完AB所有链之后，pA=pB=None。所以循环就结束，因此这个方法能够解决所有的两链表情况。
